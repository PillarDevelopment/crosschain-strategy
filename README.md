# Cross_chain Strategy
## Installation
* Copy-paste `env.template` to `.env`
  * Fill `DEPLOYER_PRIVATE_KEY` with your wallet PK to deploy on testnet
  * Fill `PROD_DEPLOYER_PRIVATE_KEY` with your wallet PK to deploy on production
